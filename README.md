## 🧭 Visão Geral

### 📌 O desafio

Desenvolver uma aplicação simples porém estruturada profissionalmente para validação de CPF, aplicando boas práticas de engenharia de software, como separação de responsabilidades e organização modular.

---

### ⚙️ Funcionalidades

- ✅ Validação completa de CPF
- 🎯 Formatação automática (000.000.000-00)
- 🚫 Bloqueio de CPFs inválidos (ex: sequências repetidas)
- 🧩 Estrutura modular (services + utils)
- 🌐 Pronto para deploy com GitHub Pages
